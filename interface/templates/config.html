Comment obtenir votre Chat ID
  1. Créez un bot avec @BotFather
  2. Envoyez un message à votre bot
  3. Ouvrez votre navigateur et allez sur :
  4. https://api.telegram.org/bot/getUpdates
  5. Cherchez "chat" -> "id" dans la réponse
  6. Copiez le numéro (ex: 123456789)
